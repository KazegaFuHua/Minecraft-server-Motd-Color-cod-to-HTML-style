薛定谔的在线

提示

输入的内容举例play.hypixel.net的服务器motd:

             §aHypixel Network  §c[1.8-1.17]\n          §5§lSKYBLOCK CRYSTAL HOLLOWS!


输入的内容举例hub.mc-complex.com的服务器motd:

§b§m-----------§8§l[- §f§lComplex §b§lGaming §8§l-]§b§m----------\n§fQuests/Clans §8| §b#1 Pixelmon Network §8| §fCustom Plugins


该脚本将输入的所有\n全都替换为了<br>（<br>是HTML网页中的换行），所以无法显示\n


如果你觉得网页控制台输出的内容比较烦人，可以用任意文本编辑器打开§ to html color.js，将第6、7、66、67这四行整行删除(均为console.log开头)，你也可以将142行的警告console.warn整行删除


你也可以通过MC服务器的API接口获取任意服务器的motd，并使用motdtocolor()函数将他们转换为html格式输出


本预览页面使用了两个字体文件，"Minecraft"与"Minecraft-Regular"，他们并不是MC原本的字体，与MC内显示的效果略有不同
